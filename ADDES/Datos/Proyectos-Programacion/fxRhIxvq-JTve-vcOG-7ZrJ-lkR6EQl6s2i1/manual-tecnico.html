Descripción

Resolución de Triángulos ofrece una variedad de ejercicios y problemas aplicados en los que se debe encontrar las medidas y los ángulos de triángulos generados de forma aleatoria, sobre un plano cartesiano o en un escenario 3D. Es una herramienta para resolver ejercicios de triángulos aplicando los teoremas del seno, coseno, tangente y Pitágoras. Se utilizan estrategias de simulación, con el fin de mejorar el aprendizaje en la resolución de problemas, particularmente en la resolución de triángulos. Permite implementar una secuencia didáctica de utilidad para docentes y estudiantes, además sirve como mecanismo de evaluación.\\ Un elemento efectivo para superar la desmotivación estudiantil es la puesta en marcha de metodologías que involucren herramientas virtuales para el aprendizaje de las matemáticas (Jiménez, 2019), aplicaciones llamativas, prácticas, interesantes, versátiles, que capturen la atención del estudiante y lo involucre más en el proceso de aprendizaje.\\ El software educativo es una alternativa a las clases tradicionales (Niola, 2015). Con él se optimizan los recursos que brinda la tecnología, favoreciendo el desarrollo de competencias y habilidades que induzcan al estudiante a interpretar, comprender y solucionar situaciones problema y así poder llegar a soluciones concretas (Nuñez, Ávila, and Olivares, 2016).\\ Copyright © 2022.\\

Manual Técnico

Análisis

La plataforma de resolución de problemas tiene el objetivo de presentar al estudiante un conjunto de herramientas mediante las cuales puede estudiar los métodos para resolver triángulos, estos son: 1) teorema de pitágoras, 2) razones trigonométricas, 3) teoremas del seno y coseno y 4) teorema de la tangente. La plataforma presenta un menú en el cual el estudiante puede generar problemas aleatorios en cualquiera de estos cuatro temas. Al resolver estos problemas el usuario gana puntos, que estan definidos en monedas, diamantes, copas y cofres, obedeciendo al enfoque de la gamificación. Por otra parte la plataforma pretende dar una herramienta al docente para evaluar al estudiante mediante estadísticas básicas y el cálculo de la prueba T-Student. Esta plataforma se ejecuta en un navegador web.

Diseño

Para el diseño de la aplicación se crearon tres módulos diferentes. El primero es la base de datos que contiene la definición de las tablas y sus relaciones. Consiste en un entorno de base de datos relacional que se ejecuta en el motor de base de datos de PostreSQL en la versión 12.x. El otro módulo es el Backend que contiene la definición de los servicios que se ofrecen para que el usuario pueda interactual con la base de datos. Este módulo de backend se instala en el servidor mediante el motor de Node.js y la aplicación PM2. Finalmente el tercer módulo es el frontend que contiene el diseño de las interfaces de usuario que permiten al usuario configurar y manipular toda la aplicación. Este módulo se instala en el servidor con sistema operativo Ubuntu, mediante el gestor de sitios web apache.

Implementación

Se escribio el código siguiendo la arquitectura del software. Se programaron conjuntamente el módulo de la base de datos y el módulo del backend. Estos se programaron directamente sobre el servidor adjuntando los certificados SSL correspondientes con el fin de que los servicios expuestos contenga el estándar seguro (https). Posteriormente se programo el módulo de frontend en el cual se consumen los servicios expuestos en el backend.

Validación

Para la validación de la aplicación se realizaron dos pruebas piloto con estudiantes de la Universidad del Quindío, particularmente de los programas de Ciencias Naturales y Educación Ambiental y de Ingeniería Civil. Con estos dos grupos de estudiantes se desarrollo la investigación la cual tomo los datos arrojados por el análisis de la plataforma mediante el cálculo del estadístico T-Student.

Plataforma

Esta aplicación se ejecuta sobre cualquier sistema operativo que pueda desplegar un navegador web. En la fase de validación se realizaron pruebas en Android, IOS, Macintosh, Windows y Linux.

Ambiente

La plataforma será usada en el entorno educativo, particularmente o en su mayoría instituciones de educación superior en cursos donde se estudien los métodos para resolver problemas de triángulos.

A continuación se describen los pasos para instalar la aplicación

Xampp

Para ejecutar la aplicación se requiere de un servicio Apache para visualizar la aplicación Web, para ello se recomienda descargar XAMPP (desde el siguiente vínculo \url{https://www.apachefriends.org/download.html}) una aplicación que contiene un conjunto de aplicaciones para la gestión de servicios web, entre ellos apache.\\ Una vez descargada la aplicación procedemos a hacer doble clic sobre el ejecutabla y seguir los pasos. \\ \begin{enumerate} \item Clic en siguiente [Next >] cuando aparece la primera ventana. \item Clic en siguiente [Next >] en la ventana selección de componentes. \item Si su disco de instalación por defecto es el dicso C:/ puede hacer clic en siguiente [Next >] de lo contrario cambie la ruta de instalación por una deseada por usted y haga clic en siguiente [Next >] \item En la siguiente ventana verá el idioma, que por defecto esta en Inglés, hacemos clic en siguiente [Next >] \item A continuación encontraremos una pantalla indicando que todo esta listo para la instalación (Ready to Install), presionamos el botón siguiente [Next >] y luego se iniciará la instalación. \item Terminada la instalación, hacemos clic en finalizar [Finish] y se iniciará la consola de Xampp. \end{enumerate} Una vez instalado Xampp, debemos iniciarlo y hacer clic en el botón [Start] en la consola de mensajes frente a la etiqueta de Apache. Este se iniciará en los puertos $80$ y $443$ que corresponden a los puertos para la navegación de internet.

Postgres

Descargar Postgres SQL en la versión 12.xx. Al momento de escribir esta manual se encuentra en la versión 12.13. Este software requiera que sea la versión 12. Para descargar siga el siguiente vínculo \url{https://www.enterprisedb.com/downloads/postgres-postgresql-downloads}. \\ Una vez realizada la descarga haga doble clic en el instalador y siga los siguientes pasos: \begin{enumerate} \item Haga clic en el botón [Siguiente >] \item Seleccione el directorio en el cual quiere que quede instalado el software, por defecto será en C:\textbackslash Program Files\textbackslash PostgreSQL\textbackslash 12. Si tiene otro disco duro en el cual instale sus programas puede cambiar ésta ruta. Haga clic en [Siguiente >] \item En la ventana donde se muestran los componentes, haga clic en [Siguiente >] \item Se mostrará una ventana preguntando la ruta o directorio donde se almacenanaran sus datos. Si tiene un disco duro para almacenar los datos eliga la nueva ruta, de lo contrario haga clic en [Siguiente >] \item Ahora escriba la siguiente contraseña 5716f2746fdaf169c38b4b5d79f7d810 esta se ha elegido para la aplicación. Presione el botón [Siguiente >] \item En la siguiente ventana se le preguntará por el puerto, que por defecto debemos dejar en $5432$ \iten En la siguiente ventana se pregunta por opciones avanzadas, dejaremos por defecto la configuración del instalador y presionamos en [Siguiente >] \item El instalador nos mostrará un resumen de lo que se ha elegido para instalar, presionamos en [Siguiente >] \item Finalmente nos mostrará una ventana indicando que todo esta listo para instalar, presionamos en [Siguiente >] y la instalación comenzará. \item Terminada la instalación, desmarcamos la casilla de (Stack Builder ...) y hacemos clic en el botón [Terminar] \end{enumerate}

Node.js

Para instalar Node.js siga el vínculo \url{https://nodejs.org/en/download/current/} y descargue la opción para su sistema operativo. En esta caso el manual se esta escribiendo sobre Windows 10. \\ Una vez que descargue el instalador, haga clic en [Next] hasta terminar la instalación. Esta instalación no requiere de configuraciones especiales.

Pm2

Dado que la aplicación del \textbf{Backend} esta desarrollada en \textbf{Node.js} es necesario instalar un paquete que nos permita ejecutar la aplicación sin interrupciones, y que frente a un fallo, reinicie la aplicación y nos deje un log del problema o error. \\ Para instalar la aplicación, debe tener instalado Node.js en el sistema, tal como se hizo en el paso anterior.\\ Abra una consola de comandos o PowerShell con derechos de administrador y escriba el siguiente comando: npm install pm2 -g y presione la tecla [Enter].
